# DBAL

DataBase Abstraction Layer.

### Installation

The package can be installed via Composer by requiring the "artoodetoo/dbal" package in your project's composer.json.

```json
{
    "require": {
        "artoodetoo/dbal": "dev-master"
    }
}
```
